# AtendeLab
Sistema de Controle de Atendimentos Acadêmicos desenvolvido na disciplina de Fábrica de Software.

## Tecnologias utilizadas
- PHP 8.x
- MySQL
- phpMyAdmin
- HTML, CSS, Bootstrap 5
- Git e GitHub

## Como executar localmente
1. Clonar o repositório
2. Colocar a pasta no htdocs do XAMPP
3. Iniciar Apache e MySQL
4. Importar o script database/atendelab.sql no phpMyAdmin
5. Acessar http://localhost/atendelab/public/
6. Login: admin@atendelab.com | Senha: 123456

## Observação
Se necessário, gere um novo hash de senha acessando:
http://localhost/atendelab/public/gerar-hash.php
E atualize no banco via phpMyAdmin.
